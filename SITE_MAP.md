# Afro Arc Site Map

## Homepage Structure

### 1. Hero
Purpose:
Create an unforgettable first impression.

Needs:
- cinematic headline
- supporting subtext
- layered visual background
- animated atmosphere
- strong CTA buttons

### 2. Manifesto
Purpose:
Position Afro Arc as more than a company.

Needs:
- short bold copy
- high-end typography
- motion reveal

### 3. What Afro Arc Is
Purpose:
Explain the brand clearly without losing mystery.

Needs:
- short explanation
- polished layout
- premium section design

### 4. Core Divisions / Pillars
Purpose:
Show the scope of Afro Arc.

Suggested pillars:
- storytelling
- comics / manga / webtoons
- animation
- film / visual media
- music / sound
- games / interactive worlds
- culture / mythology / ideas
- future imagination / technology

### 5. Flagship World / IP
Purpose:
Introduce the main universe or worldbuilding potential.

Needs:
- headline
- short description
- dramatic visual framing
- room for future character/world assets

### 6. Vision
Purpose:
Explain the larger mission and long-term ambition.

Needs:
- emotionally resonant copy
- premium layout
- cinematic feel

### 7. Showcase / Gallery
Purpose:
Present future art, concepts, and worldbuilding material.

Needs:
- structured media area
- premium grid or horizontal flow
- strong visual rhythm

### 8. Call to Action
Purpose:
Push people toward the next step.

CTA examples:
- Explore the universe
- Join the journey
- Enter Afro Arc
- Subscribe for updates

### 9. Footer
Purpose:
End the page cleanly and elegantly.
