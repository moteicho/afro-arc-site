# Afro Arc Website Product Requirements

## Objective
Build a visually stunning, modern, dynamic Afro Arc website that feels premium, cinematic, futuristic, mythic, African-inspired, and emotionally powerful.

## Core Goal
Make visitors feel:
"This is not a normal brand. This is a universe."

## Primary Audience
- People interested in worldbuilding
- Fans of animation, manga, comics, sci-fi, fantasy, mythology, and future culture
- Creative collaborators
- Potential supporters, subscribers, and long-term fans

## Required Outcome
A homepage-first site that:
- clearly introduces Afro Arc
- feels immersive and alive
- is visually distinct
- can later grow into a full franchise/company website

## Must-Have Sections
1. Hero
2. Manifesto
3. What Afro Arc is
4. Core divisions / pillars
5. Flagship IP / world section
6. Vision / why this matters
7. Showcase / gallery
8. CTA
9. Footer

## Required Experience
- vivid
- premium
- dynamic
- animation-rich
- clean
- high-end
- memorable
- scalable

## Non-Negotiables
- no generic startup look
- no bland SaaS aesthetic
- no weak copy
- no cheap-looking effects
- no cluttered layout

## Technical Preferences
- Next.js
- React
- TypeScript
- Tailwind CSS
- strong motion library integration
- reusable components
- responsive design
- accessibility-aware motion
- SEO-friendly structure

## Done Means
- the homepage is built
- sections are complete
- motion is integrated
- layout is responsive
- visual identity feels strong
- code is clean and extendable
